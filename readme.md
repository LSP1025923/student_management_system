# 购物管理系统

## 需求
    1、支持多账户登录
    2、初始化账户余额为0
    3、提供充值接口，充值后才能消费
    4、 提供充值购物功能，可加入购物车，使用账户余额结账
    5、实现提现功能，手续费10%
    6、 实现账户间转账
    7、记录日常消费流水
    8、记录用户操作日志
    9、提供管理接口，包括添加账户、冻结账户、给用户充值等
    10、用户认证装饰器
    
    
    用户视图层展示的功能：
    0、退出
    1、注册功能
    2、登录功能
    3、充值功能
    4、转账功能
    5、提现功能
    6、查看余额
    7、查看流水
    8、购物功能
    9、查看购物车
    10、退出账号
    11、管理员功能
